# FALCON Module 4: AI-Driven Correlation & Reasoning Engine

Module 4 is an enterprise AI correlation and reasoning engine for the FALCON platform. It consumes `SecurityGraphEvent` objects generated by Module 3, reasons over graph topology and temporal sequences, and produces validated `CorrelatedSecurityIncident` objects for Module 5.

## Project Structure
The folder structure follows the requested layered Clean Architecture:
```
module4/
│
├── app/
│   ├── api/
│   ├── core/               # Dependency injection container
│   ├── models/             # Domain and contract models
│   ├── engines/            # AI reasoning engines (Temporal, Graph, Pattern, Sequence, etc.)
│   ├── repositories/       # Thread-safe in-memory storage
│   ├── services/           # Workflow orchestration service
│   ├── validators/         # Input integrity validation pipeline
│   ├── configuration/      # Config and features setting
│   ├── exceptions/         # Enterprise-grade exceptions
│   └── main.py             # FastAPI endpoints and startup
│
├── tests/                  # Pytest unit and e2e integration suite
│
├── examples/               # Example client runner
│
├── requirements.txt
└── README.md
```

## Setup and Installation

1. Create a virtual environment and activate it:
   ```bash
   python -m venv venv
   source venv/bin/activate  # Or venv\Scripts\activate on Windows
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Running the API Server

Start the FastAPI application using `uvicorn`:
```bash
uvicorn app.main:app --host 127.0.0.1 --port 8000 --reload
```

The API will expose endpoints at:
- **POST** `/api/v1/correlate`: Accept single or list of `SecurityGraphEvents` to return a `CorrelatedSecurityIncident`.
- **GET** `/api/v1/incidents`: Retrieve all correlated incidents.
- **GET** `/api/v1/incidents/{incident_id}`: Retrieve a specific incident.

## Running Tests

Verify the correctness of all validation and AI correlation logic using `pytest`:
```bash
python -m pytest tests/ -v
```

## Running the Sample Client

With the FastAPI server running, test the pipeline using the included client script:
```bash
python examples/run_correlation_client.py
```
